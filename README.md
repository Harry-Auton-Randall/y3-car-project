This is a simple racing game which I made for a University module. The player can race against AI-controlled opponents on a handful of tracks, and the game saves the player's best times.

CONTROLS - KEYBOARD

WASD - move (W and S for acceleration/braking, A and D for steering)\
Arrow keys - look around with the camera\
T - reset your car to the centre of the track (intended for if you get stuck, but can be done at any time)\
Esc - pause/unpause

CONTROLS - CONTROLLER

Right trigger - accelerate\
Left trigger - brake/reverse\
Left stick - steer\
Right stick - rotate camera\
"East button" (B for xbox, O for playstation, etc) - reset\
Start button - pause/unpause
- Note: The game's menus can only be navigated with a mouse cursor

TRACK SELECT REQUIREMENTS

This game allows the player to customise the amount of laps, amount of racers and starting position of the player for each race.\
The requirements for each of these is as follows:
- Lap count - must be at least 1
- amount of racers - must be between 2 and the "max racers" listed under the track name (inclusive)
- Player starting position - must be between 1 and the amount of racers (inclusive)
